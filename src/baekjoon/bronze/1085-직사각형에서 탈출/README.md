## 문제

https://www.acmicpc.net/problem/1085

`17분` 소요

## 풀이

처음에 문제가 이해가 안되어서 찾아보았다.

`(w, h)` 가 좌표 상에서 `직사각형` 을 나타낸다는 것이었고,  
`(x, y)` 가 `(w, h)` 안에 찍히는데,  
`직사각형` 의 변 중에서 가장 가까운 변을 고르는 문제이었습니댜.
