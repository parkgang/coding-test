[백준](https://boj.kr/) 에서 푼 문제를 관리합니다.

`단계` , `분류` , `Class` 와 같은 분류는 웹 사이트 상에서 찾도록 하고 해당 repo에서는 `코드` 와 `해설` 만 관리합니다.

```
baekjoon
└─[ 티어 ]
  └─[ "id-{문제 이름}" ]
    ├─{ 언어 별 코드 }
    ├─assets // 해설을 위한 자료들
    └─README.md // 해설
```

`파일 명` 의 경우 아래의 기준으로 분리합니다.

| 분류                                | 파일 명             |
| ----------------------------------- | ------------------- |
| Best 정답                           | `main`              |
| 다른 정답                           | `case{개수}-{설명}` |
| 정답은 구하지만 기타 이유로 틀린 것 | `try{횟수}-{설명}`  |
